# まだ株・奥義継承スクリーナー

月足で順張り・日足で逆張りの長期投資手法を機械化したスクリーナー。

## Phase 1 (現状)

日経225の25日乖離率を毎日監視し、「3か月ぶりの-10%タッチ」をシグナルとして出力します。

## セットアップ

```bash
pip install -e ".[dev]"
```

## 使い方

```bash
# 日経暴落シグナルの監視
python -m src.watch_runner
```

## 設定

`conditions.yaml` の閾値を書き換えるだけでロジックは変わりません。

## フェーズ

- **Phase 1 (完了)**: 日経乖離率 + 3か月ルール + コンソール/CSV出力
- **Phase 2 (予定)**: J-Quants業績フィルタ + 月足トレンド + /pool コマンド
- **Phase 3 (予定)**: バックテスト + 回収率検証
